1.2 排列

排列问题可以分为：线排列、圆排列、重排列。如他们的名字一样，线排列即把元素排成一条线（首尾不重叠），圆排列即排成一个环（首尾重叠）、重排列涉及到如果对一个重集内的所有元素做排列。⚠️注意，排列问题是有先后之分的。

Definition 1.2.1: (线排列) 设集合A={a1,a2,...,an}\mathbf{A} = \{a_1,a_2,...,a_n\}A={a1​,a2​,...,an​}. 从这n个元素中取出r个(r≤nr \leq nr≤n),按照一定次序排列起来，称为集合A\mathbf{A}A的r−r-r−排列/r−r-r−有序子集。集合A\mathbf{A}A的r−r-r−排列的所有方法个数记为P(n,r)P(n,r)P(n,r)。

Theorem 1.2.1: 对于正整数n,r,r≤nn, r, r \leq nn,r,r≤n,有:

P(n,r)=n(n−1)(n−r+1)=n!(n−r)!P(n,r) = n(n-1)(n-r+1) = \frac{n!}{(n-r)!} P(n,r)=n(n−1)(n−r+1)=(n−r)!n!​

Corollary: 当n≥r≥2n \geq r \geq 2n≥r≥2时，有：

P(n,r)=nP(n−1,r−1)=rP(n−1,r−1)+P(n−1,r)P(n,r) = nP(n-1,r-1) = rP(n-1,r-1) + P(n-1,r) P(n,r)=nP(n−1,r−1)=rP(n−1,r−1)+P(n−1,r)

证明思路：在 nnn 个元素中取出并按顺序排成 rrr 个（即 P(n,r)P(n,r)P(n,r)）——我们对一个“特别元素”（比如第 nnn 个）是否被选用进行分类。

  1. 情形 A：特别元素被选用

    • 它必须放在这 rrr 个位置中的某一个：有 rrr 种放置方式；
    • 剩下 r−1r-1r−1 个位置由其余 n−1n-1n−1 个元素按顺序填充：有 P(n−1,r−1)P(n-1,r-1)P(n−1,r−1) 种；
    • 合计：r⋅P(n−1,r−1)r\cdot P(n-1,r-1)r⋅P(n−1,r−1)。

  2. 情形 B：特别元素未被选用

    • 全部 rrr 个位置都从其余 n−1n-1n−1 个元素里产生：有 P(n−1,r)P(n-1,r)P(n−1,r) 种。

两类互斥且穷尽，因此

P(n,r)=r P(n−1,r−1)⏟选用特别元素+P(n−1,r)⏟不选特别元素.P(n,r)=\underbrace{r\,P(n-1,r-1)}_{\text{选用特别元素}}+\underbrace{P(n-1,r)}_{\text{不选特别元素}}. P(n,r)=选用特别元素rP(n−1,r−1)​​+不选特别元素P(n−1,r)​​.

证毕。

  2. 捆绑元素，要求一个元素紧跟另一个的前/后/某一特定位置, 或者逆向思维两个元素要求不满足XX条件。
